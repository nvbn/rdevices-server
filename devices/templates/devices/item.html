How-to register methods?

1) Create device declaration file, for example:

from rdclient import Device, method


class Calc(Device):

    @method('int', x='int', y='int')
    def sum(self, x, y):
        return int(x) + int(y)

    @method('None', a='int')
    def do_nothing(self, a):
        pass

    class Meta:
        uuid = '{{ device.uuid }}'

2) Run rdclient, like:

rdclient {{ device.name|slugify }}.py
